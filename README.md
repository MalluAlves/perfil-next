# Perfil Profissional - Maria Luiza Alves

Este repositório contém o código-fonte de uma página de perfil profissional e portfólio desenvolvida como parte de uma atividade prática académica na **Universidade Federal Rural do Semi-Árido (UFERSA)**. 

O objetivo principal do projeto foi realizar a migração de uma estrutura estática para o ecossistema moderno do **Next.js** utilizando o framework de estilização **Tailwind CSS**.

## 🚀 Tecnologias Utilizadas

* **Next.js 15** (com App Router)
* **React** & **TypeScript**
* **Tailwind CSS** (Estilização baseada em classes utilitárias)

## 🛠️ Funcionalidades e Aprendizados Práticos

* **Layout Responsivo:** Uso de Grid (`grid md:grid-cols-3`) e Flexbox do Tailwind para garantir uma visualização perfeita tanto em computadores quanto em dispositivos móveis.
* **Componentes Nativos do Next.js:** Implementação do componente `<Image />` para carregamento otimizado de arquivos estáticos.
* **Sintaxe React/JSX:** Adaptação e correção de propriedades do HTML tradicional para o padrão do React (como o uso obrigatório de `className`).
